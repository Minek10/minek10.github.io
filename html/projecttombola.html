| Tombola : Le Jeu |

Idée du projet

Lorsque j'étais enfant, je jouais avec mes cousin/e(s) au jeu de la tombola.

Le but du jeu ?
Chaque joueur perçois une carte avec le même nombre de numéros pouvant aller de 1 a 90. Un joueur effectue un tirage à chaque tour d'un numéro. Dès qu'un joueur possède la numéro tiré sur sa carte, il barre ce numéro. Le premier joueur avec tout les numéros barrés gagne la partie. Digitaliser ce jeu nous à tous permis de rejouer à ce jeu ... de manière plus moderne dirons nous.

Comment l'utiliser

Chaque joueur, génère une carte sur son telephone, quand un numéro sort, il suffit d'appuyer dessus. Avec un autre dispositif (smartphone, téléphone, télévision), effectuer la génération d'un nombre

Outils et languages utilisés
Outils : Visual Studio Code
Languages : HTML, Bootstrap SASS, JS
