| Pendu Pokémon |

Idée du projet

Un excercice durant ma 1ère année de bachelier m'avait été demandé, un pendu ! Le professeur laissant au choix le thème, je choisis Pokémon mon RPG préféré de jeunesse
La génération du pokémon à deviner se base sur un fichier json reprenant les créatures jusque la 3ème génération

Comment l'utiliser

Après avoir renseigné un nom, il faudra deviner la créature qui se cache derrière les tirets soit avec le clavier ou les lettres à l'écran.

Outils et languages utilisés
Outils : Visual Studio Code
Languages : HTML, SASS, JS
